# 소개

## 개요

Tspec은 [TypeScript](https://www.typescriptlang.org/) 타입으로부터 API 문서를 생성하는 라이브러리입니다.

TypeScript 타입을 자동으로 파싱하여 최신 [OpenAPI Specification](https://swagger.io/specification/)을 생성하고, 아름다운 [Swagger UI](https://swagger.io/tools/swagger-ui/)로 제공합니다.


## 왜 Tspec인가?

- **Code First**: TypeScript 타입과 JSDoc을 기반으로 OpenAPI Specification을 생성합니다.
- **쉬운 학습**: 새로운 OpenAPI 스펙 문법을 배울 필요가 없습니다. TypeScript 타입만 사용하면 됩니다.
- **쉬운 사용**: 몇 줄의 코드만으로 OpenAPI Specification을 생성할 수 있습니다.
- **유연함**: 원하는 프레임워크를 사용할 수 있습니다. 프레임워크에 종속되지 않습니다.
- **바로 동작**: CommonJS와 ESM을 기본 지원합니다. 트랜스파일러(ts-node, babel, swc)가 필요 없습니다.
